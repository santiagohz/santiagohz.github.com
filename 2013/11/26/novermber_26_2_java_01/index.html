几点Java基础知识（1）

晚上继续学习Android基础，看到了Intent传递变量。对于ADT的内容更进一步认识，隐约中似懂非懂了MainActivity.java，AndroidManifest.xml，activity_main.xml之间的关系。

但是对于教程中的Java语法还是不太清楚。虽然之前在大学的时候学过几节Java编程，但是现在留在脑海的也就project，class，public,static这几个关键词。网上搜索了一点资料，再次学习一下Java的基本知识：

  • Jave的package和import机制，参考这篇Blog。
  • public和private的区别：public声明的内容在任何地方都可以调用；private声明对的内容只有在当前类中被调用。（虽然我还是不太清楚什么时候用，暂时都先用public吧）
  • Java8种基本数据类型：byte、int、long、double、char、boolean、float、short。
  • String不是基本数据类型。具体使用方法可以参考这篇Blog。
  • Java对大小写敏感，类、方法和变量命名的时候需要注意。基本命名规范如下：
    • 包名：多单词组成是全部为小写字母。例如package com.santigaohz.test1;。
    • 类名和接口名：单词首字母都需要大写，例如public class MainActivity。
    • 变量名和函数名：驼峰命名，例如protected void onCreate(Bundle savedInstanceState)
    • 常量名：所有字母都大写。多个单词组成时，用下划线（“_”）连接。如MAX_VALUE
  • Java语句需要以;结尾。
  • Debug：设置断点->单步执行->观察变量。

今天先到这，明天继续。
